Web Security


How to perform an SQL-Injection?

Inject SQL Commands into un-validated Form Fields to trigger execution of SQL Events on Server

What is the General Idea behind SQL-Injections?

Mixing up Data and Code

How can one prevent SQL-Injection?

Validate Input
  • Check given Input (if it has the expected Form)
  • Sanitize Input by modifying it such that the result is correctly formed

What does XSS stand for?

Cross Site Scripting

What's the target of XSS?

Executing JS Code via victim's Browser (on website)

What are the Attacker's Goals?

  • Alter Page Content
  • Track Events (mouse clicks, keystrokes)
  • Issue and read web requests and replies
  • Read and Set Cookies
  • Manipulate AJAX Requests

What Types of XSS exist?

  • Reflected

    Data in HTTP parameters is directly executed by server (exploit by manipulating URLs)

  • Persistent

    Attackers Data is stored on server and returned on page request (exploit using comments for example

What does CRSF stand for?

Cross Site Request Forgery

What are the differences between XSS and CRSF?

  • XSS: The Attacker tries to controll what the website sends
  • CRSF: The Attacker tries to controll what the client's browser sends
