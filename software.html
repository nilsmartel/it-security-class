Software Security


What is a Vulnerability?

Exploitable Flaw in a software or a system
(not every bug is a vulnerability)

Where do they come from?

  • Flaws in Design
  • Implementation errors
  • (Mis-) Configuration
  • Mis-use by User

How to prevent Vulnerabilitys?

  • Security Aware...
    • design
    • Implementation
    • operation

What are root causes of Vulnerabilitys (in Softwaredesign)

  • Inconsistency between Design and Implementation
  • Mixing of Control-Structures and Data

What is an Exploit?

Piece of Software that takes advantage of an vulerability

What are common exploit phases?

  • Injection of Code/Data
  • Manipulation of Information/ Control Flow
  • Privilege elevation

How to exploit a Stack Overflow?

  • overwrite current and previous stack frames
  • Manipulation of local Variables and return address

What can be done by manipulation local Variables?

Redirection of controll flow by variable Assignment

What can be done by manipulation of return address?

  • Injection of executable Code (aka. Shellcode)
  • redirection of return address to injected code

What are the dangers of a Read Overflow?

might leak secret Information.
