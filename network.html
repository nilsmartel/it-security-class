Network Security


What are the Layers of the TCP/IP Model?

  • Application
  • Transport
  • Internet
  • Link

What is the function of the...

  • Application Layer

    Interacting with Network Applications

  • Transport Layer

    Delivery and multiplexing of data to network applications

  • Internet Layer

    Addressing and transfer of data between gateway and hosts

  • Link Layer

    Interface with control of physical Devices

What are example protocols of the...

  • Application Layer
    • HTTP
    • FTP
  • Transport Layer
    • TCP
    • UDP
  • Internet Layer
    • IP
    • ICMP
  • Link Layer
    • PPP
    • ARP
