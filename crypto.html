What is Kerckhoff's Principle?

The Security of a Cryptographic Scheme must depend only on the key, not on the secrecy of the underlying algorithms

What is the difference between Assymetric Encryption and Digital Signatures?

Assymetric Encryption:
- Use public Key to encrypt plaintext
- Use private Key to decrypt ciphertext
Digital Signatures
- use private key to sign a message
- use public key ot verify a signature
